# Check-VAT_VIES

* Check VAT number on VIES - http://ec.europa.eu/taxation_customs/vies/
* Show web page
* Return results
* Print web page

## Usage

```powershell
PS >  Check-VAT_VIES -TIN DE99999999999

Date                NIP           User     Result
----                ---           ----     ------
2017-08-18 16:47:04 DE99999999999 user1    True
```
